# AI Lead Personalization Skill

## Description
Generates personalized outreach messages for real estate leads using AI.

## Actions
1. Analyze lead information (source, price, location, description)
2. Generate personalized first-contact message
3. Adapt tone based on lead type (buyer, seller, investor)
4. Create multiple message variations for A/B testing

## Usage
```
Generate personalized outreach message for this lead:
Type: Buyer
Location: Austin, TX
Price Range: $300k-$500k
Description: Looking for 3BR house near good schools
Tone: Professional but friendly
```

## Output
Personalized message string + message variations array
